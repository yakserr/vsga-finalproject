## Detail of Folder

| Name   | Description                                                 |
| ------ | ----------------------------------------------------------- |
| assets | provides a root-level directory for web server helper files |
| src    | provides a root-level directory for PHP source code files   |
| vendor | provides a root-level directory for additional vendor files |
